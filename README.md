# Vorotex K06 Control

Утилита для управления клавиатурой Vorotex K06 через системный трей. Позволяет быстро переключать профили раскладки.

## Описание

Программа создает иконку в системном трее, через которую можно загружать различные конфигурационные файлы (`.yaml`) в клавиатуру, используя утилиту `ch57x-keyboard-tool.exe`.

## Требования

- Windows
- Python 3.8+
- `ch57x-keyboard-tool.exe` (должен находиться в папке с программой)

## Установка

1.  Клонируйте репозиторий или скачайте файлы.
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Использование

1.  Запустите скрипт:
    ```bash
    python tray_switcher.pyw
    ```
    Или просто дважды кликните по `tray_switcher.pyw`, если `.pyw` файлы ассоциированы с `pythonw.exe`.

2.  В трее появится сине-черная иконка.
3.  Нажмите правой кнопкой мыши для выбора профиля:
    - **Работа**: загружает `work.yaml`
    - **Стрелки**: загружает `arrows.yaml`
    - **Медиа**: загружает `media.yaml`

## Настройка

Список профилей можно отредактировать в файле `tray_switcher.pyw` в секции `НАСТРОЙКИ`:

```python
PROFILES = [
    ("Работа", "work.yaml"),
    ("Стрелки", "arrows.yaml"),
    ("Медиа", "media.yaml")
]
```
