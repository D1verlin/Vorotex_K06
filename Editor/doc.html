Available Actions Доступные Действия

This document describes all available actions that can be assigned to keys and knobs on CH57x macro keyboards. Этот документ описывает все доступные действия, которые могут быть назначены на клавиши и ручки макро-клавиатур CH57x.

Overview Обзор

Each key or knob on your macro keyboard can be programmed to perform different actions. Actions are defined in the YAML configuration file and can be: Каждая клавиша или ручка может быть запрограммирована на выполнение различных действий. Действия определяются в конфигурационном файле YAML и могут быть:

  • Keyboard actions: emulate keypresses, key combinations, and sequences Действия клавиатуры: эмуляция нажатий, комбинаций и последовательностей
  • Mouse actions: control mouse movement, clicks, and scrolling Действия мыши: управление движением, кликами и прокруткой
  • Media keys: control media playback and system functions Медиа клавиши: управление воспроизведением и системными функциями

Keyboard Actions Действия Клавиатуры

Simple Keys Простые клавиши

Press a single key without modifiers: Нажатие одной клавиши без модификаторов:

  • a — 'A letter' key (see note below)клавиша 'A' (см. примечание ниже)
  • slash — '/' keyклавиша '/'
  • enter
  • esc
  • space
  • f1

To see a complete list of supported key names, run: Чтобы увидеть полный список поддерживаемых имен клавиш, запустите: 

./ch57x-keyboard-tool show-keys

HID Codes

It is also possible to emulate press of key for which no name is given, use HID code in angle brackets like this: Также возможно эмулировать нажатие клавиши, для которой нет имени, используя HID-код в угловых скобках:

  • <101>

See USB HID Usage Tables (section 10) for HID usage code list. См. Таблицы использования USB HID (раздел 10) для списка кодов.

Important / Важно When you use letters, you specify key to press, not character to produce. If you use keyboard layout other than QWERTY, pressing 'a' key may in fact produce another character, not 'a'. При использовании букв вы указываете клавишу для нажатия, а не символ. Если вы используете раскладку отличную от QWERTY, нажатие клавиши 'a' может произвести другой символ.

Key Combinations Комбинации клавиш

Use modifiers with keys by joining them with a hyphen (-): Используйте модификаторы с клавишами, соединяя их дефисом (-):

  • ctrl-c
  • cmd-v
  • alt-tab
  • shift-a
  • ctrl-alt-del

Modifiers Модификаторы

You can combine multiple modifiers:Можно комбинировать несколько модификаторов: ctrl-shift-a, cmd-alt-esc.

Key Sequences (Chords) Последовательности (Аккорды)

Execute multiple key presses in sequence by separating them with commas (,). Up to five combinations may be used: Выполнение нескольких нажатий последовательно, разделяя их запятыми (,). Можно использовать до 5 комбинаций:

  • h,e,l,l,o — types "hello"печатает "hello"
  • ctrl-a,ctrl-c — select all, then copyвыделить всё, затем копировать
  • win-r,c,m,d,enter — open Run dialog and type "cmd"открыть "Выполнить" и ввести "cmd"
Note for 3x1 keyboards / Заметка для 3x1 Key modifiers are only supported for the first key in a sequence. You can use ctrl-alt-del,1,2 but not ctrl-alt-del,alt-1,2. Модификаторы клавиш поддерживаются только для первой клавиши в последовательности. Вы можете использовать ctrl-alt-del,1,2, но не ctrl-alt-del,alt-1,2.

Mouse Actions Действия Мыши

Mouse actions allow you to control the mouse cursor and buttons directly. Supported buttons: left, right, middle. Combination: left+right. Позволяют управлять курсором и кнопками. Кнопки: left, right, middle. Комбинация: left+right.

Supported modifiers: ctrl, shift, alt. Модификаторы: ctrl, shift, alt.

Note / Примечание Unlike keyboard actions, mouse actions only support these three modifiers (no Win/Cmd key). В отличие от клавиш клавиатуры, действия мыши поддерживают только три модификатора (без клавиши Win/Cmd).

Mouse Click Клик Мыши

  • click(left) — left button clickлевый клик
  • click(left+right) — two buttons clickклик двумя кнопками
  • ctrl-click(left) — modifier clickклик с модификатором
  • shift-click(right)

Aliases: click (left), rclick (right), mclick (middle).

Mouse Move Движение Мыши

Move cursor by relative offset. Range: -128 to 127 pixels in each direction. Сдвиг курсора относительно. Диапазон: -128 до 127 пикселей в каждом направлении.

  • move(10,0) — 10 pixels right10 пикселей вправо
  • move(-10,0) — 10 pixels left10 пикселей влево
  • move(0,10) — 10 pixels down10 пикселей вниз
  • move(5,5) — 5 right and 5 down5 вправо и 5 вниз
  • ctrl-move(10,5) — move holding Ctrlдвижение с зажатым Ctrl

Mouse Drag Перетаскивание (Drag)

Drag with mouse buttons pressed: Перетаскивание с зажатыми кнопками:

  • drag(left,10,0) — drag 10px right with left buttonтянуть 10px вправо левой кнопкой
  • drag(right,-5,5) — drag with right buttonтянуть правой кнопкой
  • drag(left+right,0,10)
  • shift-drag(left,10,10)

Mouse Wheel Колесо Мыши

Rotate the mouse wheel. Range: -128 to 127. Вращение колеса. Диапазон: -128 до 127.

  • wheel(1) — rotate up by 1 unitна 1 единицу вверх
  • wheel(-1) — rotate down by 1 unitна 1 единицу вниз
  • wheel(3)
  • ctrl-wheel(1) — rotate holding Ctrl (zooms)с Ctrl (зум)
Note / Примечание Unit given to wheel action doesn't correspond to anything, it is neither line nor pixel. And it is also not linear: wheel(100) scrolls just 3 times more than wheel(1). Единица в действии wheel не соответствует ни строкам, ни пикселям. Шкала нелинейна: wheel(100) прокручивает лишь в 3 раза больше, чем wheel(1).

Aliases: wheelup is wheel(1), wheeldown is wheel(-1).

Notes & Help Заметки и Помощь

Custom Keyboard Layouts Пользовательские Раскладки

When using custom keyboard layouts (like Dvorak), you must specify the key as it appears on a QWERTY layout, not the character it produces. The keyboard emulates physical key presses, not character input. При использовании кастомных раскладок (например, Дворак), указывайте клавишу так, как она выглядит на QWERTY. Клавиатура эмулирует физическое нажатие клавиши, а не ввод символа.

Getting Help Получение помощи

  • Run to see all supported key names: Запустите, чтобы увидеть все имена клавиш:
    ./ch57x-keyboard-tool show-keys
  • Run to check your configuration: Запустите для проверки конфигурации:
    ./ch57x-keyboard-tool validate your-config.yaml
  • See example-mapping.yaml for a complete configuration example. См. example-mapping.yaml для полного примера конфигурации.
